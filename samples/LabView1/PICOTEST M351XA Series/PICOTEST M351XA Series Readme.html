PICOTEST M351XA Instrument Driver Readme

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: PICOTEST CORP.
Supported Language(s): LabVIEW 8.2
Supported Model(s): M3510A, M3511A
Model(s) Tested: M3510A, M3511A
Interface(s): USB, IEEE 488.2 (GPIB)
Firmware Revision(s) Tested: 1.03-03-01-13

Certified: Yes
NI Supported: Yes
Source Code Available: Yes

Driver Revision: 1.0
Original Release Date: 10/30/2009
Current Revision Date: 10/30/2009

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

The support software required by the driver.
VISA 4.0 or later ( http://ni.com/visa )

Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

To report issues or provide feedback about this instrument driver, please send an email to instrument.drivers@ni.com. or to e_service@picotest.com.tw.
